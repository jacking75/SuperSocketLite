# Tutorial: SuperSocketLite

SuperSocketLite는 **네트워크 애플리케이션 서버**를 *쉽게 개발*할 수 있도록 도와주는 프레임워크입니다.
복잡한 **소켓 통신** 부분을 추상화하여 개발자가 *비즈니스 로직*에 집중할 수 있게 해줍니다.
TCP/UDP 통신, 다양한 프로토콜 처리, 세션 관리 등을 지원합니다.
